descriptionDie größere Schwester der Leichten Schwarm\-rakete, die Schwere Schwarm\-rakete, ist ebenfalls ein paranidisches Waffensystem, das darauf abzielt, defensive Raketen\-abwehr\-maßnahmen eines gut gerüsteten Ziels zu überwinden. Wie ihr kleineres Gegen\-stück setzt auch die Schwere Schwarm\-rakete auf eine zweistufige Lösung. Eine größere Primärstufe besteht aus einer äußeren Hülle, die im Inneren mehrere kleinere Raketen birgt. Die äußere Hülle sorgt dafür, dass sie mit standardisierten Raketen\-werfern kompatibel ist, und die im Inneren getragenen Sub-Raketen werden nur wenige Sekunden nach dem Start freigesetzt. Zu diesem Zeitpunkt zerstreuen sie sich und folgen einem Infrarot-Lenkungs\-laser vom Start\-schiff zum Ziel. Die Nutzung des Lasers macht das System etwas billiger, was dazu beiträgt, die Kosten für mehrere Lenkungs\-sektionen auszugleichen, macht die Waffe aber auch anfällig für Infrarot-Täusch\-körper wie Leucht\-raketen.\n\nDie Sub-Raketen haben recht starke, hoch\-explosive Spreng\-köpfe hinter einer Teladianium\-scheibe. Wird sie ausgelöst, formt sich das Teladianium zu einer Art Pfeil. Dieser Pfeil dringt in das Ziel ein, wo er in große, verheerende Splitter aus fast geschmol\-zenem Metall zerspringt und enormen Schaden anrichtet.
