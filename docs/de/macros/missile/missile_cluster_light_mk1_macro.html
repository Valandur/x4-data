descriptionWie der Name schon sagt, verzichtet ein Cluster-System auf eine mächtige Einzel\-ladung und setzt statt\-dessen mehrere kleinere Sprengsätze gleichzeitig frei, die mehrere Ziele auf einmal beschädigen können. Der an jedem Einzelziel verursachte Schaden ist naturgemäß deutlich geringer als bei einer Einzel\-ladung, aber mit einer ausreichend starken Sub-Munition kann ein Pilot ein feindliches Ziel schnell außer Gefecht setzen, indem er die wichtigsten Komponenten ausschaltet.\n\nWie die Dumbfire-basierten Geschosse verfügt auch die Cluster-basierte Waffe nicht über die Möglichkeit, Veränderungen in der Flugbahn des Ziels zu kompensieren. Die Zieldaten aus den Computer\-systemen an Bord des Schiffes werden genutzt, um kurz vor Abschuss der Waffe einen Zeitzünder zu programmieren. Sobald der Zeitzünder abgelaufen ist, idealerweise kurz vor dem Einschlag der Rakete, wird eine Vielzahl von intelligenten Mini\-bomben oder "Bomblets" über das Zielgebiet verteilt. Diese verfügen über eine recht fort\-schritt\-liche Software zur Zielsuche, die sie in die Lage versetzt, die anfälligeren Teile der exponierten Systeme eines feindlichen Akteurs aufzuspüren.
